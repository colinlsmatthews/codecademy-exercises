HTML CSS Cheatsheet

Table Tags

Tag Name Description
<table> Table Defines a table; wraps all other table elements
<thead> Table Head The set of rows comprising the header of a table; wraps header cells
<th> Table Header Defines a header cell in a table
<tbody> Table Body The set of rows comprising the body of a table
<tr> Table Row Defines a row in a table; wraps table cells
<td> Table Data Defines a cell in a table
<tfoot> Table Foot The set of rows forming the footer of a table
<caption> Caption Defines a caption for a table

Table Attributes

Attribute Name Description
colspan Column Span Defines how many rows a <td> element will span
rowspan Row Span Defines how many columns a <td> element will span
